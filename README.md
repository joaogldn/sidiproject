# Sistema de Ponto Eletrônico SiDi 

## 📋 Sobre o Projeto

Sistema desenvolvido durante a **Residência Onboard Rise Up** no **Porto Digital**.
Desafio proposto pela SiDi para criar uma solução que resolvesse o problema de inconsistências nos horários dos funcionários no quesito de marcar ponto, agrupando em um só aplicativo todas as funcionalidades necessárias para o colaborador.

### ✨ Funcionalidades

- **Marcação de Ponto**: Registro de entrada/saída com horário em tempo real
- **Abono de Faltas**: Solicitação de abono com justificativa e anexos
- **Corrigir Ponto**: Requisição de correção de marcações equivocadas
- **Histórico de Pontos**: Visualização de marcações anteriores
- **Notificações por Email**: Configuração de alertas personalizados
- **Alteração de Senha**: Atualização segura de credenciais

## 🚀 Tecnologias Utilizadas

- React 18.x
- React Router DOM
- CSS3 com variáveis personalizadas
- TypeScript

## 📦 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

- [Node.js](https://nodejs.org/) (versão 16 ou superior)
- [npm](https://www.npmjs.com/) (geralmente vem com o Node.js)
- [Git](https://git-scm.com/) (opcional, para clonar o repositório)

## 🔧 Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/sidiproject.git
```

### 2. Acesse a pasta do projeto
```bash
git clone https://github.com/seu-usuario/sidiproject.git
```

### 3. Instale as dependências
```bash
npm install
```

### 4. Execute o projeto
```bash
npm run dev
```
O projeto será aberto automaticamente em http://localhost:5173
